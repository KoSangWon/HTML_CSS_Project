# 접근성을 고려한 배민 문방구 홈페이지

## 프로젝트 개요

적응형 홈페이지인 배민 문방구를 반응형으로 만들었습니다. 따라서 하나의 HTML 파일로 뷰포트의 크기에 따라 레이아웃이 변하는 것을 볼 수 있습니다. 또한 WCAG 2.1버전에 있는 지침들을 근거로 웹 접근성을 향상시켰습니다.

## 역할 분담

윤다은 고상원 오재원 장지훈
1. 윤다은 : 헤더 영역
2. 고상원 : 홈페이지 메인 영역
3. 장지훈 : 상품 상세 페이지 메인 영역
4. 오재원 : 로그인 페이지, 푸터 영역

## 코딩 컨벤션

Sass 사용 시 BEM 컨벤션을 사용하였습니다.
